# Reasons I Love You

A personal appreciation page built for someone special. Each visit presents a riddle to solve before revealing a reason why she's loved.

## What This Is

An interactive web page where riddles unlock personal messages. Features include:
- Riddle-based unlock system with optional clues
- Optional "sass" additions for playful humor
- Mobile-optimized, app-like interface
- Smooth animations and transitions

Built with vanilla HTML, CSS, and JavaScript.

## Deployment

Currently hosted via GitHub Pages at: `https://rbs92.github.io/repipage/`

To update, simply push changes to the main branch and GitHub Pages will redeploy automatically.
